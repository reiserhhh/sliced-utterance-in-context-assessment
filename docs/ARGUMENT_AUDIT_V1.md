# SUICA 論証監査 v1 — v4原稿の論理×裏付けの主張別検品

作成: 2026-07-07(第12検品の前段)。対象: SUICA_JPA_MANUSCRIPT_V4.md(v4.1、式・付録A実装後)。
方法: 原稿の主要主張を一件ずつ取り出し、(a) **論理** — 前提から結論が実際に従うか、
(b) **裏付け** — 台帳row・一次記録・数値と一致するか、を独立に検査する。
原則: **主張は推論だけでは立たない。裏付けの種別を明示し、種別を超えた強さで書かない。**

種別: 導出(証明済み代数)/ 監査済実証(敵対的監査を通過)/ 探索的実証(T1)/
Sim(反事実世界)/ 文献 / 推論。判定: ✅堅牢 / 🔧修正済み(v4.2)/ ⚠️残存弱点。
