# SUICA スイカ語彙対照表 v1(用語集)

作成: 2026-07-07。ユーザー指摘「電池・楽器は不自然、スイカ的な名詞にすべき」
への回答。方針: **日本語の報告・JP向け文書はスイカ語彙**を使う。
**英語の方法論文書は標準語(battery / instrument 等)を保持**——査読者の
共通語なので。この表がその橋。各行 = 英語標準語 / 旧直訳 / スイカ語彙 /
それが実際に指すもの。

## 使用例(現在のプロジェクト状態をスイカ語彙だけで)

SUICA は、玉を切らずに叩いて中身を当てる検査法である。一玉=一人の
テキスト全体。審査表 v4 の項目は仕分け済みで、果肉系(皮をまたいでも
変わらない味: 一人称・アポストロフィ省略・否定・罵り)が4つ、残りは
皮系と混合系。採点の前にシール剥がしを通し、検品は11回、採点者のミスを
13件見つけた。取り置き玉は最後の1玉が蔵に残り、玉割りの封印状は公開の
蔵に貼ってある。配合表(玉割り#2 の主予測子候補)は凍結済み。そして
種模様——並みの味を引いた後に残る、その玉だけの並び——は半年置いても
見分けがつく。二つと同じ種模様の玉は、理論上も実測上も、ない。

## 運用規則

1. 私(AI)の日本語報告・ユーザー向け文書・将来の日本語版マニュアルは
   この語彙を使う。
2. 英語の方法論文書(THEORY/RULEBOOK/論文原稿)は battery / instrument /
   lockbox 等の標準語を保持する(査読者の共通語)。
3. 対応が曖昧になったらこの表が正。表にない新概念は、命名時にこの表へ
   追記してから使う。
