# V7 Evidence Status Lattice

## Why a lattice

SUICA evidence is not a single promotion ladder. Reproducible geometry,
scoreability, uncertainty, repeated-measurement reliability, external
construct validity, and cross-domain transport answer different questions.
Passing one coordinate cannot promote another.

## Coordinates

| Coordinate | Current admissible states | V7 current state |
| --- | --- | --- |
| Geometry | absent / technical / replicated within declared cohort | technical and internally replicated candidates |
| Scoreability | unavailable / frozen profile / support refusal active | frozen reference-relative profile implemented |
| Conditional uncertainty | absent / partial / joint nested | protocol implemented; empirical joint estimates pending |
| Repeated-measurement reliability | untested / design ready / estimated | untested; future-data preflight implemented |
| External construct validity | untested / protocol ready / anchored / independently replicated | untested; pre-label protocol implemented |
| Cross-domain transport | none / local reference only / paired alignment / pretrained alignment | local-reference-only protocol; paired evidence absent |

## Combination rule

The public status is the vector of coordinate states, never the maximum state.
For example, internally replicated geometry plus untested reliability remains
an internally replicated text-geometry candidate. It is not a reliable
personality score.

## Present V7 claim

The strongest current claim is:

OPERATOR_INDEXED_RELATIVE_TEXT_GEOMETRY_WITHIN_DECLARED_DOMAIN

This means V7 can define and replay an author-relative geometry under a frozen
observation operator, representation, runtime, reference cohort, and support
rule. It does not mean that anonymous coordinates are personality factors,
clinical scores, temporally stable traits, culture-free universals, or market
predictors.

## Migration rule

A new domain such as market text may use the formula and procedure, but must
fit a local reference population and preserve its own operator index. Source
norms, landmarks, and numerical score meanings do not transfer without paired
or otherwise identified alignment evidence. Outcome labels remain outside the
measurement stage.
