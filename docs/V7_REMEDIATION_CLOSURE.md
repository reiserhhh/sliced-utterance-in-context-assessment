# SUICA V7 Remediation Closure

## Scope

This document records how the V7 methodological gaps are handled. `Closed`
means that a mathematical safeguard, simulation calibration, or executable
refusal rule now exists. It never means that a human personality construct has
been established from PANDORA text.

| Gap | Resolution | Current status | Remaining boundary |
| --- | --- | --- | --- |
| Target ambiguity / run provenance | Schema-valid estimand manifests, runtime and input hashes, artifact inventories, and identifier-free cohort commitments. | `IMPLEMENTED` | A frozen implementation can still measure only its declared text object. |
| Raw shared covariance confounded by observed conditions | W2 simulations compare raw and train-fitted context-adjusted subspaces. Only the latter is allowed to recover the planted shared component. | `CALIBRATED` | This validates a synthetic estimand, not a human author core. |
| Linear transport mistaken for all structure | W2 includes nonlinear worlds where relative-distance geometry survives while linear transport fails. | `CALIBRATED` | Relative geometry is not a named factor or a psychological construct. |
| Temporal order treated as privileged evidence | W3 uses source-disjoint early/late panels and same-budget random source-half controls. | `TESTED` | Random halves frequently preserve as much or more geometry; time order is not a trait-retest proxy. |
| Opportunity / formatting treated as either all noise or all signal | W3 records opportunity-only and opportunity-adjusted controls separately from selection/topic. | `TESTED` | Opportunity is a material alternative layer, not a causal explanation or default subtraction target. |
| One consensus method mistaken for truth | W4 compares consensus covariance, concat-PCA, and ridge SUMCOR RGCCA against correspondence-breaking nulls. | `SUPPORTED_AS_METHOD_COMPARISON` | Both fresh cohorts support all three; their small ordering changes do not establish a universal winner. |
| Predictive rank mistaken for a number of factors | W4 rank selection reaches 8 in the first cohort and 16 after an independent grid extension. | `RANK_UNRESOLVED_HIGH_DIMENSIONAL` | Transport is a capacity criterion. It cannot identify a finite factor count; rank must stay an operator parameter. |
| Direct cross-view prediction mistaken for a shared construct | Direct Ridge is reported as an unconstrained comparator and is consistently higher than shared-subspace R2. | `BOUNDARY_ENFORCED` | Better prediction does not demonstrate a common axis, author core, or psychological score. |

| No single-text error theory | Conditional uncertainty separates source clusters, model refits, reference norms, and radial-envelope support. ## Verified Numerical Facts

- W2 corrected full calibration: in the planted shared-linear world,
  train-fitted context-adjusted subspace congruence was `.997 [.995, .999]`.
  In the nonlinear world relative-distance Spearman was `.393 [.325, .459]`
  while linear transport R2 was only `.068 [-.075, .166]`. Endogenous and
  error-prone context counterexamples show that no universal residualization
  rule is identified.
- W3 corrected full: 25 independent random-half partitions were nested in the
  uncertainty analysis and all included temporal rows had observed numeric
  order. Temporal-minus-random AUC means were `-.053`, `-.026`, and `-.040`;
  all intervals crossed zero. This blocks a privileged-time or trait-change
  reading and quantifies ordinary source-partition variability.
- W4/W4b: all registered shared methods beat correspondence-breaking nulls
  (`max-T p=.01` per endpoint family). W4's best shared R2 was `.267` on 240
  authors; W4b's was `.397` on a separate 200-author cohort. The method winner
  changes across cohorts, so no method hierarchy is promoted. Direct Ridge
  remained larger (`.621`/`.630` mean R2 in W4b) but is explicitly not a
  shared-subspace estimator.
- W4b spectral diagnostic: null-adjusted cross-view covariance remains
  distributed rather than low-rank. Confirmation entropy effective rank is
  `22.75`/`22.93` for word/character representations, with `21` directions
  needed for 90% of positive excess energy. Null envelopes are calibrated
  separately at discovery n=120, calibration n=36, and confirmation n=44.
  This is a
  representation-specific capacity description, not a number of traits.
- The real-data geometry smoke scored `98.6%` of discovery, `92.2%` of
  calibration, and `79.2%` of confirmation authors. The `20.8%` confirmation
  radial-envelope refusal rate is an empirical boundary, not missing data to
  impute or permission to extrapolate. It is not a general OOD detector.
- W7 full transport calibration showed that unknown rotation makes naive
  coordinate retrieval collapse (`.0018` accuracy), while paired Procrustes
  alignment recovered it (`1.000`). The target-local procedure is reusable;
  unpaired source-to-target score meaning is not identified.

## Current Scientific Position

V7 has a reproducible **operator-indexed relative text geometry** with
technical author alignment, condition/opportunity controls, and explicit
refusal regions. It has not established a stable named factor inventory,
personality score, clinical state score, or universal cross-language latent
space. Evidence is reported as a lattice rather than a single promotion level:
geometry is implemented, while reliability, external validity, and paired
cross-domain transport remain separate future-data gates. The next legitimate
empirical upgrade is a newly designed repeated fixed/free study, not further
relabelling of PANDORA geometry.
