# V7 Theory Closure Contract

## Closed object

V7 closes an executable and refusing framework for operator-indexed relative
text geometry. It freezes the observation operator, representation, runtime,
reference population, metric, landmarks, support region, and provenance. It
calibrates when geometry, linear transport, residualization, temporal
partitioning, and cross-domain alignment are or are not identified.

## Not closed

The closure does not establish a named factor, personality construct,
reliability coefficient, clinical interpretation, culture-free coordinate, or
market predictor. Those are independent coordinates in the evidence lattice
and require future data.

## Durable closure

The machine audit verifies decisions, source/input manifests, artifact
inventories, and required protocols. A dirty repository can pass the
implementation audit but receives:

V7_THEORETICAL_CORE_CLOSED_WITH_EMPIRICAL_GATES

Only a clean, intentionally frozen source snapshot can receive:

V7_THEORETICAL_CORE_CLOSED_WITH_EMPIRICAL_GATES

No commit or tag is created automatically by the audit.
