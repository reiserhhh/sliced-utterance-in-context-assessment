# V7 Conditional Geometry Uncertainty Protocol

## Object

A geometry profile for one author is conditional on source units, a fitted
representation/aggregation runtime, and a reference population. Its uncertainty
must not be reported as one generic SEM unless these sources are modeled.

For a scalar profile functional g, retain at least:

\[
\operatorname{Var}(\hat g) =
V_{\mathrm{source}} +
V_{\mathrm{model}} +
V_{\mathrm{reference}} +
2\sum_{a<b}\operatorname{Cov}_{ab}.
\]

The three minimum components are:

1. source-cluster variability from non-overlapping author source units;
2. model-refit variability from discovery representation/aggregation refits;
3. reference-norm variability from reference-population resampling.

Rows outside the frozen radial envelope or below minimum observation support
receive a refusal, not a precision interval. This radial check is not a full
out-of-domain or local-density detector.

## Combination Rule

The default accepted total uncertainty is generated by nested joint draws:

1. resample discovery/reference rows;
2. refit the registered representation and geometry runtime;
3. resample the scored author's independent source clusters;
4. score the profile functional using that frozen draw;
5. use the empirical draw distribution for the conditional interval.

If joint nesting is infeasible, SUICA may report each component separately. It
must refuse a total SEM unless an independence approximation is explicitly
declared. That approximation is not a reliability coefficient.

## Boundary

This protocol does not estimate G-study/LST trait reliability. The repeated
participant-by-condition-by-occasion design is separately required for that
question. It also does not eliminate opportunity, domain, language or
representation drift; those conditions remain part of the support domain.
