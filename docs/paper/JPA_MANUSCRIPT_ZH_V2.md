# 作为"被选情境中之行为"的文本: 一个基于自发语言的人格评估理论

## 摘要

语言人格评估立于一项矛盾之上。其预测模型的成功主要来自话题特征——人们
**选择**写什么——而其测量实务却把话题与场合当作应予移除的滋扰方差。
我们以一个认真对待这一选择的理论化解矛盾: 一个人积累的书写,是分布于
其自选情境之上的行为; 人格经由三条通道进入文本——情境的选择、情境
不变的文体基底、以及人×情境签名。自三条公理出发,我们推得对实务有
直接后果的结果: 原始稳定性混同了文体与被冻结的情境选择(一个完全没有
个人文体的合成语料,仅凭选择即产生重测 r = .64); 统计移除情境均值是
中介变量调整,按可推导的精确数量删除个人信号,仅在命题指派下无害;
特质语言唯有通过分离通道的纯度判据方获执照。该理论解释了悬而未决的
谜题——为何话题特征预测得好、为何功能词是恒久稳健的标记、为何分数
跨平台在方向上可迁移而在水平上不可——并且禁止得足够多、足够冒险:
其自身的预注册总括检验未通过(2/7),却恰好确认了两项通道预测(第一
人称率→神经质, r = .111; 政治场合选择→开放性, r = .096)。我们将理论
置于情境选择、行为签名、代表性设计、文体计量学与个案式测量的谱系之中,
并推导据此建造的评估工具必须主张与不得主张之事。全部材料、形式证明与
封印的预注册均已公开。

**关键词:** 人格理论; 语言评估; 情境选择; 行为签名; 个人语; 心理测量

## 1. 悖论

支撑过去十年语言人格研究的两项发现,在现行实务下不可能同时成立。

其一是预测性的: 人格可从日常语言中读出,而且**最**可读的是由话题主导
的开放词汇特征——人们选择书写的活动、兴趣与场合(Schwartz et al.,
2013; Park et al., 2015; Eichstaedt et al., 2021)。开放词汇模型胜过
封闭词典,正因为它收割内容。

其二是方法性的: 在同一文献中,话题被当作污染。流程对内容作残差化、
按 subreddit 或体裁分层语料、或为了移除而建模主题——其明言的逻辑是:
必须先把"怎么写"从"写什么"中分离出来,才能干净地测量人格。

实务反驳了发现。若话题方差是滋扰,移除它应使个人信号更锐; 若话题特征
携带最好的信号,移除它应折损精度——而该领域大多把这一张力当作工程
权衡,而非理论症状。我们在1,760万条评论上,以其最强形式直接检验了
滋扰前提: 朴素条件均值中心化、收缩部分中心化、交叉拟合样本外双向固定
效应中心化。**每一个变体都降低了每一个文体测度的分离时段稳定性**
(合并 Δ = −.091, 95% CI [−.102, −.080]; 交叉拟合变体 −.05 至 −.07)。
滋扰前提不是无用,而是方向相反。

还有第二个更安静的症状。文本信度向来按问卷信度的读法来读: 分数稳定
即禀性稳定。可是,一个只不过持续光顾相同场合的书写者,在任何受场合
影响的测度上都会得到稳定分数——无论是否存在任何个人文体。在一个
**毫无**个人文体的生成语料中——文体完全由场合决定、场合由人际稳定的
偏好选出——"文体"分数的普通重测相关是 r = .64。稳定性本身对文体毫无
证明力。该领域自我理解的两件最基本工具——偏差化与信度——都默默预设了
情境的外生性: 一个从实验设计悄悄输入到"每个情境都是被选的"观察语料
中的假设。

化解之道不是选边,而是改变"噪声"是什么。人格心理学曾经走过这步棋。
当行为拒绝跨情境保持恒定时,Mischel 与 Shoda(1995)既未捍卫恒定性,
也未放弃人格; 他们把情境间的变异本身重新概念化为签名——if A then X,
if B then Y——悖论遂告消解。我们在下一层重复这步棋。语言评估所删除的
话题方差,并非人周围的误差; 在自我选择之下,它就是**人本身**,经由
删除所摧毁的那条通道得以表达。下文给出建立在这一反转之上的理论、它
蕴含的形式结果、它解释之物,以及——理论靠其禁止之物赢得信用(Meehl,
1978)——它赖以立或倒的冒险承诺,包括一次部分失败的预注册检验。

## 2. 理论背景: 五根支柱

理论并非凭空唤出; 每个承重构件都有确立的谱系。理论本质上是它们的
合取,故逐一陈述各支柱独立确立之事。

**情境选择是行为。** 人与环境相互对应,因为人非随机地**选择**自己的
环境、诱发反应、并操纵情境(Buss, 1987); 现代情境研究记录了日常情境
接触与解读的特质一致性(Rauthmann et al., 2015)。在文本语料中,每条
话语的场合与话题都是**被记录下来的选择**。丢弃它们的测量理论是在
丢弃行为——恰是评估宣称要取样的范畴。

**变异有模式,而模式是个人的。** 行为签名文献确立: 跨情境变异不是
误差而是结构——稳定的个体 if-then 剖面(Mischel & Shoda, 1995)。其
文本推论: 个人在特定情境类别内对自身基线的偏离,是与均值不同的、
可测量的签名通道。

**评估应取样生态。** Brunswik 的代表性设计主张: 泛化要求从有机体的
生态中代表性地取样情境,而非构造系统性题目集。一个人积累的自发书写
正是这样的生态样本——丰沛、非引出、产于其真实生活的情境。问卷是系统
性设计; 切片的自然文本才是代表性设计。传统所缺的,是让生态样本可用于
测量的东西: 冻结的、可审计的计分。

**个体性居于形式。** 一个世纪的文体计量——从功能词著者归属(Mosteller
& Wallace, 1963)到"人类文风组"(van Halteren et al., 2005)与个人语
共选择(Wright, 2017)——表明语言的个人独有层不成比例地由**形式**承载:
功能词、正字法习惯、形态句法共选择。心理语言学补充: 即使母语语法也
在个体间稳定地不同(Dąbrowska, 2018)。形式习惯是情境不变文体的天然
候选; 内容词汇是情境的天然载体。

**特质描述分布; 个人超出因子清单。** 整体特质理论把特质的描述面建模
为状态的密度分布(Fleeson, 2001; Fleeson & Jayawickreme, 2015)——而
切片语料正是这样一个分布,特质分数是其参数。同时,个案式传统
(Allport, 1937)、其非遍历性的现代形式化(Molenaar, 2004; Fisher et
al., 2018)与剖面分解心理测量(Furr, 2008)共同告诫: 任何个体间因子
结构都穷尽不了一个人——并给残差以可测定义: 减去常模剖面后所余之物。

五柱合取,蕴含一个特定对象: 自发文本是分布于被选情境之上、取样自本人
生态的行为; 个人信号居于选择本身、居于形式承载的不变习惯、居于模式化
的情境偏离——以及居于任何因子清单都无法捕获的残差之中。文本评估的
理论应当就是这一对象的形式化。下面给出它。

## 3. 理论

### 3.1 公理

**A1(切片是行为观察。)** 语料被切分为固定长度的话语切片; 每个切片是
某一情境中言语行为的一次观察。人由其切片的分布来表示——密度分布观的
文本应用。

**A2(情境被选择,且选择人际稳定。)** 每个切片携带其情境 r(场合/话题
壳)。在自由生态中情境是自选的,且本人的情境分布 π_u 是稳定的个人
属性(情境选择支柱——被记录,而非被推断)。

**A3(加法分解。)** 切片分数分解为 y = f_u + b_r + γ_{u,r} + e:
情境不变的个人基底 f、情境效应 b、人×情境交互 γ、切片噪声。

### 3.2 三通道——作为推论

由 A1-A3,人格恰经三条路径进入文本: 选择分布 π_u(C1)、不变基底 f_u
(C2)、γ_{u,r} 的结构(C3——文本中的行为签名)。模型中再无其他带人称
下标的项。通道不是隐喻,而是 A3 中载人项的划分。

### 3.3 形式结果

*(证明与模拟验证见补充材料 A; 先以文字陈述。)*

**定理1(复合稳定性)。** 原始个人均值测量 f_u + m_u,其中
m_u = Σ_r π_ur·b_r 为情境混合项; 既然 π_u 稳定(A2), m_u 亦稳定。
故原始分数的重测稳定性混同 C2 与被冻结的 C1: 共享情境重测协方差等于
Var(f) + Var(m) + 2Cov(f, m)。系(已演示): 即使 f ≡ 0,原始重测仍读出
r = .64。未经通道分离,稳定性不为任何特质语言发放执照。

**定理2(中心化是中介变量调整)。** 移除估计的情境均值即减去 m̂_u。
在选择-效应独立下,被移除的重测协方差恰为 Var(m) + 2Cov(f, m)(神谕
模拟: 预测 −1.335,观测 −1.332); 在人-情境耦合下,移除更甚于此,因为
被估计的情境效应吸收了选择该情境者的个人基底(估计对真实的回归斜率
在耦合 κ = .7 时为 1.40; 观测移除 −1.583,一阶预测 −1.294)。边界条件:
情境被外生指派时 m ≡ 0,中心化无害。话题控制唯在指派设计中是清洗——
在自我选择下,是按可推导数量进行的删除。

**定理3(识别要求不相交)。** 把个人的情境拆为不相交集合可消去混合-
混合协方差项(在情境效应独立与选择-效应不耦合下),故不相交集重测
在耦合交叉项以内单离果肉项——一个上界; 类别不相交变体另移除类别级
中介。由此得到操作性**纯度判据**: 唯当类别不相交稳定性可观而选择中介
份额小时,构念方可以特质语言描述。

**结果4(载荷加权精度)。** 情境异质性按构念的情境载荷向信度征税:
场合混合的熵折损情境承载构念的精度,而放过形式承载者。

**结果5(过度识别: 理论自我监察)。** 跨时段或跨语域的迁移以
√(rel₁·rel₂) 为界; 独立中介估计量必须一致; 不一致即内建的误设警报。
(它已在我们自己的数据中触发; 受影响的估计以区间报告。)

**系(比较执照)。** 分数只在其通道构成支持的半径内可比: 情境内监测;
纯度通过构念的跨主题比较; 跨语域比较须配语域别常模; 跨体制唯方向与
总体内序位——跨自由/指派/领域锁定体制的水平比较被定理2的边界结构
禁止,任何协变量调整都无法恢复之。

### 3.4 个案式之层

令常模剖面为诸构念上的总体均值; 移除之后个人的偏差剖面即其**籽纹**
(Furr, 2008)。理论为评估加上第三个描述层——位置(通道分离构念上的
分数)、构形(if-then 签名)、残差(籽纹)——并提出可证伪主张: 残差
不是噪声,而是稳定的个体性。实测: 相隔数月的时段之间,籽纹以中位
r = .505 自相关(陌生人零线 −.02; 仅用形式构念为 .684); 同时,约三分
之一的机器可识别个人信号留在因子清单之外,却仍人际稳定。两个籽纹无须
重合——这不是诗意修辞,而是指纹直觉的操作内容: 个人可被编码,却不可
被因子穷尽。

## 4. 理论解释之物

理论应比其所取代的假设更经济地回溯该领域的稳健发现。

**为何开放词汇特征预测得好。** 因为它们收割 C1。话题与场合词承载着
封闭文体词典设计上排除的选择通道; 开放词汇对封闭词典的预测优势
(r ≈ .31–.41 对 .21–.29; Schwartz et al., 2013)在本理论下主要是通道
覆盖,而非词汇分辨率。

**为何功能词是恒久的标记。** 因为它们是果肉。文体计量学的常数——归属
与稳定个人风格乘于形式之上——是从另一学科看到的 C2 通道。在我们自己
的构念清单中,通过纯度判据的恰是形式习惯(代词率、撇号省略、否定、
脏话),而每个内容词汇构念都未通过; 清单中最强的果肉纯构念是一个人类
评分者不可见(盲评 0/8)、却在未见用户上保持 r_B = .649 的正字法习惯
——个体性在形式之中,恰在文体计量学所说的位置。

**为何分数在方向上迁移而在水平上不迁移。** 定理2的边界结构蕴含: 水平
受体制约束(场合规范、命题约束、平台产出限制),而构念-锚定关系的方向
乘于 f 与 π 的结构之上。观测: 在 Reddit 上拟合的量表原样迁移到学生
作文,保持平均方向效度(r ≈ .144),与不透明嵌入迁移(≈ .153)同级,
而水平可比性失败; 且理论的两项通道预测在下述封印检验中跨语料边界获
确认。

**为何情感词率作为特质令人失望。** 情感词汇主要是切片级与时段级方差
(密度分布意义上的状态),且带重情境载荷(结果4): 其原始稳定性是情境
承载的,纯化后的稳定性接近零——我们的 tension 构念即内部演示,在纯度
判据下判定不能,作为特质已死。恰如整体特质区分对一个在错误时间尺度
上取样的状态量所预测的那样。

**为何人-情境之争在每条文本流水线中重演。** "是否对情境残差化"的
流水线选择,是在估计量内部重演1968年的论争。理论的答案与该领域最终的
答案相同: 争议中的方差经由选择与签名属于人; 当作通道来对待,以设计来
控制。

## 5. 理论禁止之物及其战绩

冒险内容,以禁令与进行中预测的形式陈述(每项主张的实证状态见表2;
全部制品公开)。

**禁止: 自我选择下的话题控制。** 如预测般被证伪,经三条独立途径
(第1节); 定理给出机制与确切代价。

**禁止: 由原始稳定性得出特质语言。** 必须先通过纯度判据; 将其应用于
我们自己的19构念清单,大部分遭降级(果肉纯4、场合承载8、混合6、判定
不能1)——包括我们此前自己描述为文体的构念。

**禁止: 跨体制的水平比较。** 无协变量可恢复之(定理2); 受试的统计
捷径(限于形式坐标)亦经验地失败,只余设计(同一人的多体制取样)一途
——我们已指定该设计研究。

**进行中的预测。** 指派情境语料是中心化转为无害的边界条件(可在任何
命题语料上检验); 果肉在任何计分器修订下都聚于形式; 关于尚未写出之
文本的哈希封印结构预测已在公开记录,静候其窗口。

**封印检验,完整报告。** 一次预注册的一次性验证开箱针对隔离标签执行
(假设、方向、资格、校正与成功规则事先公开封印)。总括规则**未通过**:
7个假设中2个达 BH-FDR q < .05,而规则要求4个。通过的两个恰是理论的
两条核心通道——第一人称率→神经质(r = .111, 95% CI [.051, .170],
q = .002)与政治场合选择→开放性(r = .096, [.036, .155], q = .006)——
而失败的五个,无一例外,是纯度判据(事后推导、追溯适用)归入场合承载
或判定不能的构念。我们将此作为纪律在运转来报告: 框架自身的入场闸门,
若在预注册时已存在,本会禁止大多数失败假设; 理论真正关涉的两条通道,
在一场旨在杀死它们的检验中幸存。

## 6. 对评估的蕴涵

**工具的形状。** 设计上两相: 自由采集(C1 存活: 选择剖面+生态文体
取样)与指派情境量表(C1 静默; 依边界条件纯化 C2)——把问卷的标准化
与表现样本的自发性相结合这一旧抱负的修正形态。分数按三层报告(位置、
构形、残差),每层携带其比较执照; 被禁止的比较由公开软件拒绝,而非
留给行文。

**可主张之事。** 特质语言仅在纯度通过之后; 跨体制唯方向与序位,水平
永不; 状态语言仅在取样支持的时间尺度上; 同一性级主张(残差层)仅对
陌生人零线。这些是化为规范的定理,不是文风上的审慎。

**认识纪律,一段话。** 因本理论主要由人类指挥下的 AI 系统构建并检验,
每个估计量在其输出可支持主张之前,须在违反理论自身公理的生成世界中
取得执照; 全部验证材料置于恰好执行一次的公开哈希封印预注册之后; 每个
实质结果都通过了拥有完全重算权的对抗性审计——11轮中捕获并修正13个
构建者错误,每个错误、其捕获、及其产生的常规均为公开记录之一部分。
