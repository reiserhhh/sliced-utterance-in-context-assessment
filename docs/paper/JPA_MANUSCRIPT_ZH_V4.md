# 被选情境中的行为之文本: Sliced Utterance In-Context Assessment(SUICA)的理论、开发与预注册首次检验

## 摘要

人在其所写之物中显露自身; 以日常语言评估人格的文献如今已具相当规模。
其中多数工作把文本"写了什么、写在哪里"当作噪声处理: 特征按话题做残差化,
语料按场域分层, 测量则常被限定在封闭的风格词典之内。我们论证, 在自我选择
的写作中, 这一习惯删除的是信号而非噪声, 因为写作者自行选择情境, 而这些
选择本身就是一致的、表达人格的行为。我们将这一观察发展为一个框架: 累积
文本被读作分布于被选情境之上的行为, 经由三条通道到达测量——情境选择、
情境不变的风格基底、以及人×情境签名。框架的承诺在 PANDORA 上构建并受检,
这是由其作者提供给我们的、含 1,760 万条 Reddit 评论的语料库(Gjurković
et al., 2021)。如预测所言, 去除情境均值使我们检查的每一个风格测度的
个人稳定性下降(合并 Δ = −.09); 区分"人所承载"与"场域所承载"稳定性的
纯度标准, 在十九个词汇构念中只放行四个作为特质候选, 且四者皆为形式习惯。
随后, 一次密封的、一次性的预注册检验确认了七个方向假设中的两个——
第一人称率预测神经质(r = .111), 政治场域选择预测开放性(r = .096)——
而总括标准未获通过; 幸存者恰是框架的两条通道级预测。同一批用户上的
探索性拟合梯度, 将这些无拟合分数与监督模型(平均 r = .27)相对照。我们
讨论测量——不同于预测——有资格主张什么。

**关键词:** 人格评估; 言语行为; 情境选择; 行为签名; 个人语; 语言分析

## 引言

过去二十年, 人们在日常生活中产生的语言已成为人格数据的严肃来源。
Pennebaker 与 King(1999)表明风格性词语使用跨时间与文体是可靠的;
Yarkoni(2010)在近 700 名博主上绘制了大五的词级相关地图; Schwartz
等(2013)分析了 75,000 名社交媒体用户的约 7 亿词, 发现开放词表语言
特征以惊人的特异性还原人格、性别与年龄。基于语言的评估与自评相关、
与知情人报告一致、且数月间稳定(Park et al., 2015)。对一个长期渴望
问卷之外行为学替代方案的领域, 这些是令人鼓舞的结果, 它们也确实鼓励了
从大规模流行病学到临床筛查的应用。

较少受到注意的, 是贯穿这类工作的一个方法论习惯。因为人格被理解为应当
跨情境成立之物, 而文本随其恰好谈论的内容变化极大, 研究者例行地把话题
与情境方差当作干扰处理。这些操作很熟悉: 在话题模型上做残差化、分析前
按论坛或文体分层语料、或者恰恰因为内容词看似被环境污染而把测量限定于
封闭风格词典。这一直觉表面上合理。但我们将指出, 它与该领域自身的发现
处于安静的冲突之中, 因为预测得最好的开放词表特征多半是话题性的
(Schwartz et al., 2013; Eichstaedt et al., 2021)。倘若人们写什么大体
只是环绕人的噪声, 删除它本应有所帮助。

我们最近在 PANDORA(Gjurković et al., 2021)上对这一噪声假定做了直接
检验——这是其作者为人格研究编纂、并直接提供给我们的语料库, 含约一万名
用户的 1,760 万条 Reddit 评论。我们施加了所能构造的最强形式的情境
中心化, 包括以交叉拟合做样本外估计的双向固定效应。每一个变体都降低了
我们此后称为"分离时段稳定性"的量——即从一个人早期写作与后期写作分别
计算的分数之间的相关, 两者零文本共享; 这一下降对我们检查的每个风格
测度都成立, 跨测度合并的损失为 Δ = −.09(95% CI [−.10, −.08])。这个
结果只在我们考虑"噪声"方差从何而来之前显得悖谬。在观察性语料中没有人
指派话题。写作者选择他们的场域与主题, 选择不断重复, 而重复的选择恰恰
是评估本应捕捉的那种一致的、有心理表达力的行为。在这样的数据里对情境
做统计控制, 等于把人格到达纸面的一条通路调整掉了。

我们提议的重新定向有先例可循。当跨情境的行为一致性难以找到时, 一种
回应是捍卫特质, 另一种是抛弃特质; 而最终立住的解决两者都不是。Mischel
与 Shoda(1995)论证, 情境变异性本身是有结构的, 稳定的"如果-那么"
剖面是人格的签名而非围绕它的误差。我们的提议把同一教训应用于言语行为。
语言研究者所去除的情境关联方差, 不是围绕人的误差。在自我选择的语料中,
它的相当部分看来就是这个人, 经由一条会被去除操作摧毁的通道表达出来。

本文因此做出三项相连的贡献。第一项是理论的: 我们依托情境选择研究、
行为签名文献、Brunswik 的代表性设计、风格计量学与个体内分布研究,
组装"情境选择是个人性的"这一论证, 并陈述一个带形式后果的框架——包括
标准操作(中心化、信度解释)何时误导、何时安全的说明。第二项是一个
检验法: 一套把人的累积文本读作被选情境中之行为的计分程序, 由固定的
公开词表与场域选择剖面构成, 每个构念在被称为"特质"之前都被强制通过
纯度检查。第三项是有监管链的证据: 在隔离标签上对七个方向假设的预注册
密封检验(框架只部分通过), 以及把检验法与拟合于同一语料的预测模型
相对照的基准分析。通篇我们力求所主张不超出证据所许可; 框架禁止若干
常见操作, 而它自身在这些禁令之下的成绩, 是论证的一部分。

## 情境选择为何是个人性的

四条研究线索——没有一条是我们的——汇聚于同一前提: 在*某处*做某事,
对做事者是有信息的。

第一条是情境文献。Buss(1987)区分了选择、唤起与操纵这三种使人与环境
趋于对应的机制, 并把对环境的非随机选择列于首位。当代情境研究已把这一
论点定量化: 在经验取样工作中, 人们报告自己身处的情境及其解读方式,
有规律地追踪其特质(Rauthmann et al., 2015; Rauthmann & Sherman,
2023)。外向者更多出现在社交情境中, 部分是因为他们安排自己如此。在
文本语料中这一机制不是推断而是被记录的: 每个帖子的场域与主题都是一次
被记录在案的选择行为。把这份记录当作噪声, 丢弃的恰是人格心理学花了
数十年论证其重要性的那类行为。

第二条线索关乎情境之内发生的事。Shoda 等(1994)在反复出现的夏令营
情境中观察儿童, 发现情境-行为随附关系的剖面——"如果-那么"签名——即使
在平均行为不稳定时也稳定且有区分力。文本的类比很容易陈述: 写作者进入
某类场域时其风格如何移动, 本身可能就是一个稳定的个人模式, 有别于其
平均风格。我们把这类签名当作第三条测量通道, 并注意它们需要自己的零
模型, 因为在某个场域里发生某种风格移动, 对所有在那里发帖的人都是
常态性的。

第三, 有一个古老的论证说评估应当对生态取样。Brunswik 的代表性设计
主张, 泛化要求从有机体的实际环境中取样刺激与情境, 而非构造系统化的
题目集(Dhami et al., 2004)。这样看, 一个人日常生活的累积写作不是
标准化测验的穷亲戚, 而是更具代表性的样本, 其精神更接近 Mehl 对日常
言语的自然主义声音取样(Mehl et al., 2006), 或接近观察者据以判断
陌生人、准确度令人意外的表达行为薄片(Ambady & Rosenthal, 1992)。
生态传统在文本上一直缺的是标准化计分; 文本检验法必须补上的是这个
缺口, 而不是行为的短缺。

第四, 一个世纪的风格计量学指明了语言中个人层所在的位置。自 Mosteller
与 Wallace(1963)以来的归属研究反复发现, 辨识个体的与其说是他们谈论
什么, 不如说是形式上的小习惯: 功能词、标点与正字法习惯、偏好的句式。
van Halteren 等(2005)据此类规律性论证存在"人类风格组", 而关于个人语
的法庭语言学工作指向同一方向(Wright, 2017)。心理语言学补充道, 稳定
的个体差异甚至延伸到母语语法知识本身(Dąbrowska, 2018)。如果情境
不变的个人风格存在于文本之中, 它应当不成比例地栖居于形式; 相反, 内容
词汇是情境的天然载体。我们将看到, 这一分野几乎正是通道分离测量所
还原出来的东西。

第五项考虑为全局定框。Fleeson(2001)表明, 一个人的状态最好被描述为
一个密度分布, 特质即该分布的参数; 文本切片的语料正是这样一个分布,
取样自生活而非由题目诱出。而从 Allport(1937)直到其在非遍历性论证中
的现代形式化(Molenaar, 2004; Fisher et al., 2018), 个体化传统警告:
个体间结构并不能穷尽个体。Furr(2008)把剖面分解为常模成分与独特成分,
给了这一警告一个可测量的形式, 我们把它采纳为框架的残差层。

合起来, 这些文献暗示对文本语料的一种特定读法: 它是行为, 分布于此人
选择的情境之上, 取样自此人的生态; 其中个人性经由选择、经由形式承载的
习惯、经由有模式的情境移动而显现, 并显现于因子清单捕捉不到的残差之中。
下面的框架, 是把这一读法弄得足够精确以至于可能出错的尝试。

## 作为情境化行为的文本框架

框架需要三个假定, 无一奇特。第一, 语料可以被读作行为观察的样本。我们
把每个人的文本切成固定 token 长度的切片, 把每个切片当作一次情境中言语
行为的观察, 于是一个人由观察的分布而非单一文档来代表。这是特质的密度
分布观(Fleeson, 2001)向文本的移调, 其重要性与在彼处相同: 分布的
均值、离散与随附关系是不同的量, 把它们压扁会丢失信息。第二, 每个切片
携带一个情境标签, 而在自由生态中情境是被选择的。选择分布本身看来就
相当稳定: 在我们的开发语料中, 用户场域分布跨时间的相似性, 在任何风格
测度进入之前, 就以接近 .84 的曲线下面积区分同人对与异人对(.50 为
随机, 1.00 为完全分离)。第三, 我们假定切片分数可加性地分解为个人
基底、情境效应、人×情境交互与噪声。我们把这当作工作近似而非形而上学
承诺; 其若干可检验后果在下文受检, 而我们自己的一个估计器存在的主要
目的, 就是在该近似失效时拉响警报。

用符号写出: 对于人物 u 在场域 r(i) 写下的切片 i,

    y_ui = f_u + b_r(i) + γ_u,r(i) + ε_ui,        r(i) ~ π_u,

其中 f_u 是此人的情境不变基底, b_r 是场域 r 的风格偏移, γ_u,r 是
人×场域签名, ε_ui 是噪声, π_u 是此人的场域选择分布。一个人的原始
构念分数是切片均值 ȳ_u = f_u + m_u + γ̄_u + ε̄_u, 其混合项

    m_u = Σ_r π̂_ur b_r

只是选择剖面的函数; π̂_ur 是 u 的切片落入场域 r 的实现份额。

在这些假定下, 人格恰好可以经由三条路线到达文本分数: 经由选择分布
本身、经由情境不变的基底、经由交互项的结构。我们称之为选择、基底与
签名通道。为它们命名的要点不是分类学而是记账: 每一个关于"风格"的主张,
都隐含地是一个关于观察到的协方差由哪条通道承载的主张, 而在研究者例行
施加的操作之下, 这些通道的行为并不相同。

三个命题使之具体。证明连同足以重算每个量的估计器定义, 载于附录A; 这里用文字与关键恒等式陈述内容。

*命题1(稳定性是复合物)。* 一个人切片分数的均值, 估计的是其基底与
一个按选择加权的情境效应混合项之和。因为选择分布稳定, 混合项也稳定,
于是原始分数的重测相关同时反映两条通道。形式地说, 在份额稳定时, 原始分数的分离时段协方差是 Var(f) + Var(m) + 2 Cov(f, m), 外加签名与噪声项(附录A.2)。这一实践后果容易被低估。在
一个被构造为完全不含个人风格——只有带风格的场域与个人稳定的场域偏好
——的模拟语料中, 普通的分离时段重测相关仍达到 .64。单独来看, 文本
测度的高"信度"并不表明存在风格特质; 它只表明某种与人相连的东西是
稳定的, 而那东西可能是此人对场域的口味。

*命题2(中心化是中介调整)。* 从分数中减去估计的情境均值, 去除的是
混合项, 连同经由选择传行的全部个人信号。在选择与情境效应独立时, 被去除的重测协方差恰为 Var(m) + 2 Cov(f, m)(附录A.3), 而已知真值的模拟几乎精确地复现它(预测
−1.335, 观察 −1.332, 协方差单位)。当选择与情境效应耦合时——只要人们
寻找适合自己风格的场域, 二者就耦合——估计的情境效应还会额外吸收基底
本身的一部分, 中心化去除的就多于闭式的预测; 在我们的模拟中, 估计
情境效应对真实情境效应的回归斜率在强耦合下达到 1.40。有用的部分是
边界条件: 当情境是被指派而非被选择时, 混合项为零, 中心化无害。我们
相信, 这正是同一操作在命题作文研究中表现得合情合理、却在社交媒体语料
中悄悄损害测量的原因。

*命题3(辨识需要分离)。* 如果稳定性是复合物, 分离通道就要求比较在
不相交情境集合上计算的分数, 使共享的情境效应无法承载相关。在单个场域
层面分割可去除大部分混合项; 在场域类别层面分割, 则可去除经由类别内
相关情境效应运作的中介。由此我们导出特质语言的纯度标准: 只有当一个
构念在完全不同类别的场域之间保持稳定(跨类别分离稳定性), 且该稳定性
中只有一小部分可归于稳定的场域选择而非风格(低选择中介份额)时, 它才
被描述为风格特质。该份额以协方差记账估计: 中介份额 = 1 − Cov(f̂_e, f̂_l)/Cov_raw, 其中 f̂ = ȳ − m̂, 而混合估计 m̂ 交叉拟合(场域效应在其他人身上估计; 附录A.4)。我们使用的阈值(分离稳定性至少 .15, 中介份额低于
.30)是操作性约定而非导出常数, 我们如实标注。

另有两个元素补全框架。第一个是残差层。沿 Furr(2008)的分解, 我们从
每个人的构念分数向量中减去常模剖面, 把剩下的——独特剖面——当作自身
即可测量的量。理论动机来自个体化传统与非遍历性论证(Molenaar, 2004;
Fisher et al., 2018): 不应指望个体间结构穷尽个体。经验上它确实没有:
独特剖面在相隔数月的时段间以中位数 .50 与自身相关, 而陌生人配对基线
本质上为零; 仅在形式类构念上计算时, 该相关升至 .68。另一迹象指向同一
方向: 现代文本嵌入可回收的人物辨识信号中, 约三分之一完全落在我们的
构念清单之外, 却自身跨时段稳定。个体的可编码性看来远超任何固定因子
清单所能捕捉——这正是个体化传统一直所说的, 如今附上了一个数字。

第二个元素是一组比较许可。因为通道对平台、提示与语言的改变反应不同,
一个分数只在声明的半径内挣得可比性, 且半径因构念家族而异。固定情境内
的个体内监测是要求最低的情形; 跨主题比较要求纯度; 跨语域比较要求语域
特定的常模; 而跨体制(自由 vs 指派 vs 领域锁定写作)的分数水平比较
完全不被许可, 因为命题2蕴含没有任何统计调整能使它们等值。我们发现把
这些许可编码进发布软件是有用的: 软件直接拒绝越权比较, 而不是把问题
留给脚注。

## 检验法的开发

检验法在公开语料上开发并接受压力测试, 每一次分析、修正与撤回都记录于
公开的主张台账; 此处做摘要, 程序细节置于补充材料。主语料是 PANDORA
(Gjurković et al., 2021), 我们从其作者处直接获得; 开发分析使用其中
在互不重叠时间窗内拥有足够文本的 3,183 名用户。指派情境的对照是
Pennebaker 与 King(1999)的意识流作文语料: 学生按固定指示连续写作,
话题与场域由研究者而非写作者设定。两个语料库都有长久的发表史, 我们
按其原名与原始条款使用; 我们自己收集的唯一未发表数据(一个小型市场
论坛语料)不参与此处报告的任何分析。验证标签从一开始就被隔离在以公开
仓库初始提交哈希密封的预注册之后, 只在下文报告的密封检验中被读取
恰好一次。

### 从文本到分数: 程序概略

由于此前的论证一直抽象, 走一遍实际计算的东西可能有帮助; 确切参数与
代码在补充材料。计分始于把一个人在每个场域内的评论串接起来, 切成连续
的 128-token 窗口(token 大致即一个词)。短于 24 token 的窗口被丢弃;
写作者明确自称人格标签的窗口("作为一个 INTJ……")同样被丢弃, 因为
测量没有资格从纸面上抄答案。每个幸存切片按固定词表计分。词表短小、
公开、版本锁定。例如第一人称分数就是第一人称形式的 token 百分比: 在
"I think my plan might work, but you should try it first" 一句中,
十二个 token 有两个(I, my)是第一人称, 于是该切片得分为每 100 token
16.7。一个人在某构念上的分数是其切片的平均; 复合构念是这类比率的固定
加权和。没有任何东西按人拟合; 修订任何词表都要求在冻结数据上演示修订
究竟改变了什么。

选择剖面从切片的来源而非其言说内容计算。每个场域按一次性在开发用户上
构建的固定"场域→类别"映射, 归属十二个内容类别之一(政治与新闻、游戏、
体育等); 一个人的剖面是其评论在类别上的分布, 以相对于总体的方式表达。
一个把 40% 评论放在政治场域的人——总体平均接近 10%——无论其写作风格
如何, 都有很高的政治选择分数。

稳定性与纯度通过在时间上分割每个人的写作来计算。我们取最早与最晚的
部分, 至少相隔 90 天, 分别计分, 再跨人相关两个分数; 这就是分离时段
稳定性。至于纯度, 同一分割再算两次: 一次只用两部分中同类别的场域,
一次只用完全不同类别的场域。稳定性在异类别分割中幸存的构念, 是人在
承载; 在那里塌掉的构念, 原是场域在承载。

### 塑造量表组的两项开发结果

两项开发结果——均在验证标签仍然密封期间获得——决定了量表组(battery)
的最终形态。

命题2预测: 从自我选择的文本中去除情境均值, 应当降低而非改善个人分数
的稳定性。事实如此。在初始量表组的五个风格测度上, 条件均值中心化无一
例外地降低了分离时段稳定性, 合并变化为 −.09(95% CI [−.10, −.08]);
任意收缩水平的部分中心化行为相同; 而我们所能构造的最强版本——以样本外
估计的双向固定效应中心化——在第一次尝试中的自估计泄漏被发现并修正之后,
仍使每个测度损失 .05 到 .07。在指派情境一侧, 同样的操作没有问题, 恰如
边界条件所要求。

把命题3的纯度标准应用于十九个词汇构念的完整清单, 它们被干净地分拣了。
四个作为风格特质候选通过, 四者皆为形式习惯: 第一人称率、撇号省略、
否定、脏话。所有由内容词汇定义的构念全部不合格, 其估计选择中介份额
约在 .33 到 .61 之间; 六个为混合; 一个(情感复合)连可供分类的基础
稳定性都不足。撇号一例值得一句: 它是清单中最强的风格特质(跨类别分离
稳定性 .37), 却对人不可见——盲评中人类评定者只能以随机水平辨认它
(8 人中 0 人), 而该构念在构建时从未见过的用户上保有 .65 的交叉验证
稳定性。书面语言中的个体性, 看来不成比例地栖居于小的机械习惯——这
恰是风格计量传统本会预测的; 而内容词汇的行为正如其所是: 一份关于
写作者去往何处的记录。与这一分野一致, 在一半用户上拟合、迁移到另一半
的字符 n-gram 共选择轴, 增加了与词级量表组不冗余的二十一维稳定信号,
其中约十五维是形式类的。

## 密封的验证性检验

双向意义上最强的证据, 都来自预注册的开封。在被隔离的大五标签(合格
用户 n = 1,058)上的七个方向假设与一个次级桥接假设, 连同合格性规则、
Benjamini-Hochberg 校正、先验检验力计算、以及"七中至少四个确认"的
成功规则, 在任何标签被读取之前于公开场合固定; 分析脚本本身经对抗性
审计后以哈希锁定, 只运行一次。总括规则未获通过: 七个假设中两个在校正
后幸存。幸存的两个, 恰是从框架通道最直接推出的两个: 第一人称率预测
神经质(r = .111, 95% CI [.051, .170], q = .002), 政治场域选择预测
开放性(r = .096, [.036, .155], q = .006)。没有任何显著效应与其预
注册方向相反。五个失败无一例外, 都是关于事后导出的纯度标准归为场域
承载或不可分类的构念的假设; 若该标准在预注册时已存在, 它本会把其中
大多数挡在验证性检验之外。我们把失败作为失败报告。开封所许可的东西
刻意狭窄: 两条通道级方向主张, 效应量小, 在一个总体中。它更广地说明
的是: 框架的入场规则与其验证记录指向同一方向。

## 同一语料库上的测量对预测

对任何新检验法, "与既有之物相比如何"都是公正的问题; 而在 PANDORA 上
这一比较可以异常直接, 因为语料库的作者们发表了监督基线——拟合于大五
标签的、tf-idf 加权 n-gram 上的岭回归与套索模型——并把代码与预测值
随数据一起提供。我们从该官方包出发, 把他们的官方大五基线复现到与
发布值之差 Δr ≤ .0014: 其 n-gram 模型五特质平均 r = .246(n = 1,402);
加入预测 MBTI 与九型特征的增强模型平均 r = .293(n = 1,386)。

这一比较必须小心阅读, 因为两种事业回答不同的问题(Yarkoni & Westfall,
2017)。预测模型可以拟合任何有帮助的东西; 它按留出数据上的精度受审,
其系数对"在测量什么"不负任何承诺。测量在看到效标数据之前固定其构念与
方向, 按其分数意味着什么受审。为把两者放上同一地面, 我们在与密封检验
完全相同的用户(n = 1,058, 相同折叠)上运行了一个探索性拟合梯度, 从
完全拟合走到零拟合(表1)。tf-idf 一元词上的岭回归——官方基线的模型
家族——在这些用户上达到平均 r = .272, 说明我们的合格性规则并未选出一个
特别容易或特别困难的队列。把同样的拟合限制在检验法的十六个冻结的、
预注册的特征上(四个风格构念、十一个场域选择轴、选择熵), 平均降到
.090, 增益集中在框架所说个人信号栖居之处——神经质(r = .166)与开放性
(r = .105)。把拟合完全去掉, 剩下的就是密封检验的预注册单构念
(r = .111 与 .096)。这些分析复用的是在验证性开封时已经开封的标签;
它们在构造上就是探索性的, 并如此记录于台账。

他人事后发现。

## 讨论

这个框架始于解释一个令人不适的结果的尝试, 其价值应当以它还能组织多少
其他东西来评判。一旦把文本读作被选情境中的行为, 文献中若干立定的特征
便自然随之而来。开放词表方法对封闭词典的预测优势(Schwartz et al.,
2013; Kern et al., 2014)成为预期之事, 因为话题特征承载着风格词典按
构造排除的选择通道; 按这一读法, 优势反映的是通道覆盖多于词汇分辨率,
而我们自己的拟合梯度在单一语料库内部复现了这一模式: 打开词表使平均
相关相对已承诺特征集大约变为三倍(表1)。功能词作为个体标记数十年的
稳健——从作者归属到 Pennebaker 的研究纲领——同样是预期之事, 因为形式
习惯正是情境不变基底的栖居之处。语言模型与词典在平台之间水平迁移差、
而序关系以衰减形式幸存这一常见观察, 与框架的边界结构吻合: 水平受体制
约束, 方向可以旅行。而情感词率作为特质测度的反复令人失望, 在这里有一
个简单的描述: 它们负载情境沉重, 表现为在错误时间尺度上取样的状态——
密度分布观正会如此提示。

这一说明也禁止事情, 而禁令是它最实用的产出。它禁止在自我选择语料中把
话题与场域中心化当作默认操作, 并指明使同一操作变得安全的设计替代
(指派情境)。它禁止仅凭信度做特质解释, 代之以一个操作性纯度检查——
我们自己的清单大部分没有通过它。它禁止跨自由、指派、领域锁定写作比较
分数水平——每当平台上训练的模型被用于别处, 现行实践都在隐含地做这一
比较。倘若一项设计良好的研究表明, 在清楚自我选择的语料中情境中心化
改善了分离时段的个人稳定性, 或表明仅靠统计调整就实现了跨体制的水平
可比性, 我们将认为框架已受损。

若干局限为我们的主张划界。全部特质级证据来自一个平台加一个命题作文
语料, 皆为英语, 出自写作量很大的人; 合格性规则丢弃轻量写作者, 此处
的一切对他们无话可说。语料不含人口学信息, 因此无法检查跨年龄、性别或
设备的测量不变性。已确认的效应很小(r ≈ .10-.11), 远低于任何个体级
决策所需, 我们不做任何应用主张。总括的预注册标准未获通过; 尽管失败的
模式富于信息, 它是用事后构建的标准诊断的——该诊断是关于下一项研究的
假设, 不是对这一项的挽救。最后, 这一纲领在很大程度上由人类指导下的
AI 系统执行。我们把这当作实行更严而非更松治理的理由: 估计器在违反
框架自身假定的生成世界上接受资格审查, 验证性材料置于一次性公开封印
之后, 而拥有完全重算权的对抗性审计在十一轮中抓出十三个真实错误, 全部
在案。审计轨迹是证据的一部分, 其意义如同望远镜的校准记录是观测的一
部分。

有两项研究会决定性地推动这一说明。第一项是个体内、多体制设计: 同一批
参与者在自由与指派情境中写作, 各两次, 用两种语言, 并配以问卷与知情人
锚点。这一个设计同时检验跨体制的水平等值(我们目前禁止的那个比较)、
月尺度以下的状态结构、对话语域、以及基于形式的通道的跨语言普遍性;
其完整规格已公开。第二项已以弱形式在运行: 关于尚未写出的文本的结构性
预测, 预先以哈希密封。两者遵循的都是产出当前结果的同一政策——把框架
弄得容易被杀死, 然后报告它死了没有。

人格评估长期把标准化测验与自发样本当作分属不同传统之物, 一者可计分,
一者丰富。一个人的累积写作提示这一区分并非根本。人们写什么、选择在
哪里写、以及无论如何都固执存在的小机械习惯, 全都是行为; 合起来读,
在尊重"哪条通道承载哪种信号"的规则之下, 它们产出一种计分上标准化、
来源上生态化的评估。眼前的框架是把这一读法弄精确的一种方式; 而到
目前为止, 它只在自己说会幸存的地方幸存了自己的检验。


## 附录A: 模型、证明与估计器定义

**A.1 模型与记法。** 对于人物 u 在场域 r(i) 写下的切片 i,

    y_ui = f_u + b_r(i) + γ_u,r(i) + ε_ui,        r(i) ~ π_u。

f_u 为情境不变基底, b_r 为场域风格偏移, γ_u,r 为人×场域签名, ε_ui 为
噪声, π_u 为场域选择分布。除非另行说明, 场域效应均值为零且跨场域独立;
γ 与 ε 均值为零, 与其余独立, 且跨时段独立。原始分数是切片均值

    ȳ_u = f_u + m_u + γ̄_u + ε̄_u,        m_u := Σ_r π̂_ur b_r,

其中 π̂_ur 是 u 的切片落入场域 r 的实现份额; m_u 只依赖选择剖面。以下
所有稳定性量, 都是零文本共享的早期与后期两时段分数之间、跨人物的
协方差(或相关)。

**A.2 命题1(稳定性是复合物)。** 当两时段场域集合重叠且份额稳定时,
m_u^(e) ≈ m_u^(l); 在 A.1 下展开 ȳ 并利用 γ、ε 的跨时段独立性, 得

    Cov(ȳ^e, ȳ^l) = Var(f) + Var(m) + 2 Cov(f, m) + [签名项]。

因此原始与共享场域的重测相关携带全部混合方差。果肉为零的模拟世界
(f ≡ 0, 带风格的场域, 个人稳定的偏好)使其量级具体化: 原始重测
r = .64, 共享场域 r = .43; 同一世界上 A.4 的分离估计量读出 −.07。

**A.3 命题2(中心化是中介调整)。** 选择中介人→文本(f_u → π_u →
场域 → y)。场域中心化把 y_ui 换成 y_ui − b̂_r(i), 从 ȳ_u 中移除
m̂_u。当选择独立于场域效应(π ⊥ b)且 b̂ 以样本外(交叉拟合)估计时,
移除在期望意义下精确, 分离时段协方差的变化为

    Δ Cov(重测) = − [ Var(m) + 2 Cov(f, m) ]  ≤ 0

(只要混合个人稳定且非负耦合)。使用先知(真实)场域效应时, 该恒等式
在模拟中复现至观察 −1.332 对预测 −1.335(协方差单位)。在耦合之下——
人们寻找适合自己风格的场域——b̂ 连 f 的一部分也吸收(b̂ 对 b 的回归
斜率在强耦合下达 1.40), 中心化移除的就多于一阶恒等式所述。边界条件:
指派情境下 π 跨人相同, 故 Var(m) = Cov(f, m) = 0, 中心化只移除估计
噪声。

**A.4 命题3(辨识需要分离)与纯度标准。** 设场域集合 A_u(早期)与
B_u(后期)不相交, 场域效应均值为零且跨场域独立, 且 π ⊥ b。则
Cov(m^A, m^B) = 0, 且

    Cov(ȳ^A_e, ȳ^B_l) = Var(f) + Cov(f, m^A) + Cov(f, m^B),

即分离集合估计量把 Var(f) 单离出来, 只差基底×混合交叉项: 当
Cov(f, m) = 0 时精确, 正耦合下偏向上界。耦合下有三条幸存路径, 均予
明确处理: (i) 上式两个交叉项; (ii) 同一内容类别内相关的场域效应——
它们跨"不相交"场域重开中介通路, 是跨类别变体的动机; (iii) 强耦合下的
残余混合×混合项——因为权重本身依赖 b。总中介份额的协方差记账估计
量为

    mediated_total = [Var(m) + 2 Cov(f, m)] / Cov_raw
                   = 1 − Cov(f̂_e, f̂_l) / Cov_raw,        f̂ := ȳ − m̂,

其中 m̂ 交叉拟合(场域效应在其他人身上估计; 早期版本的自估计泄漏已
发现并修正, 交叉拟合要求现为常设方针)。在蓄意违反假定的生成世界中,
mediated_total 在强耦合下向上偏(记录在案的失败世界中为 .541 对目标
.426), 故在真实数据上作为中介的上界来读; 等价地, 1 − mediated_total
是基底份额的下界, 跨类别分离稳定性提供配对的偏上界之臂, 两者夹住
真值。正文的操作性纯度标准——跨类别分离稳定性至少 .15 且中介份额
低于 .30——就是把这些估计量配上明示为约定的阈值。

**A.5 计分定义(按代码冻结原样)。** 构念的切片分数是与其固定公开
词表匹配的 token 百分比(匹配 token 数 / 切片内 token 数 × 100);
个人构念分数是切片均值; 复合构念是构念分数的固定加权和。场域按一次性
在开发用户上构建的冻结映射归入十二个内容类别。以 s_uk 记人物 u 在
类别 k 的切片份额, s̄_k 记合格队列平均份额, 则

    选择轴   a_uk = log( (s_uk + ε) / (s̄_k + ε) ),   ε = 10⁻⁴,
    选择熵   H_u  = − Σ_k s_uk log s_uk,

其中一个类别作为参照被排除, 留下十一条轴。分离时段稳定性是各人最早
与最晚写作部分(至少相隔 90 天, 零文本共享)分数之间、跨人物的
Pearson 相关; 同类别与异类别变体按正文所述约束两部分的场域。密封
检验的合格性要求每人至少 4 个场域类别、至少 12 个切片。

**A.6 过度识别检查。** 模型之下成立的三条约束被作为误设定警报监控:
(i) 沿文本量曲线的 Spearman–Brown 一致性; (ii) 对任意语域或时段对的
单因子迁移上界 Cross(e, l) ≤ √(w_e · w_l)——语域研究中的实例为
.739 ≤ √(.782 × .857) = .819; (iii) "共享对分离"对比与协方差记账两种
中介份额估计在抽样误差内的一致。不一致预警类别相关的场域效应或加法性
失效; 而恰恰观察到一例此类不一致, 才迫使 A.4 采用区间读法而非点估计。
